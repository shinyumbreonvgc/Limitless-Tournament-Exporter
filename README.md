# Limitless Tournament Exporter

A fully automated tool for exporting Pokémon VGC tournament data from LimitlessTCG.
This application analyzes tournament pages, extracts player standings, teamlists, and usage stats, then generates fully formatted .xlsx files and Showdown importable .txt files.

This project supports:
macOS (.app inside .DMG file)
Windows (.exe)

Features
Complete tournament export (standings, usage, Pokémon sets)
Top-cut usage + full meta usage stats
Fully formatted xlsx outputs
Images, sprites, Tera type colors, and more

-----------------------------------------------------------------------------------------------------------------------------------------------

Windows Installation Instructions:
Download the .exe file from the Releases page.
Locate the downloaded file in your Downloads folder.
Double-click the installer/executable to launch the program.
Windows SmartScreen Warning
Because this project is not code-signed, Windows may show a SmartScreen protection popup.

When you see:

“Windows protected your PC”

Do this:

Click "More info"
Click "Run anyway"
This is normal for unsigned applications.

-----------------------------------------------------------------------------------------------------------------------------------------------

macOS Installation Instructions:
Download LimitlessTournamentExporter.app.zip from GitHub Releases.
Open Terminal and run:

xattr -cr

(note the space at the end) Drag Limitless Tournament Exporter.app from Finder into the Terminal window so the path is added after xattr -cr.
Press Enter. Now, in Finder:
Right-click Limitless Tournament Exporter.app
Click Open
In the warning dialog, click Open

After that, they can just double-click it like a normal app.
