 # Limitless Tournament Exporter

A fully automated tool for exporting Pokémon VGC tournament data from LimitlessTCG.

This application analyzes tournament pages, extracts player standings, teamlists, and usage stats, then generates fully formatted .xlsx files and Showdown importable .txt files.


This project supports:

macOS (.app inside .zip file)

Windows (.exe)

 # Features

Complete tournament export (standings, usage, Pokémon sets)

Top-cut usage + full meta usage stats

Fully formatted xlsx outputs

Images, sprites, Tera type colors, and more
